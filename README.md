# Ananda Steel Website

A professional website for Ananda Steel export-import company specializing in agriculture commodities and steel products.

## Features
- Responsive design for all devices
- Modern and professional layout
- Product showcase for agriculture and steel
- Contact form for inquiries
- Smooth animations and interactions

## Setup
1. Upload all files to your web hosting
2. Update contact information in `index.html`
3. Add your company logo and images
4. Customize colors and content as needed

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
